## 🌟 Featured Projects

### 🎣 [react-hooks-visualized](https://github.com/purush-o7/react-hooks-visualized)
> *An interactive, themed learning platform for React Hooks & TanStack Query*

Every hook is taught through a **real-world metaphor** — spy networks, space launches, DJ booths — so the concepts *stick*, not just the syntax. Features live interactive playgrounds where you can break things and see how hooks behave in real-time.

**Built with:** Next.js 16 · React 19 · TypeScript · Tailwind CSS v4 · TanStack Query v5 · Motion

---

### 🌌 [r3f-visualized](https://github.com/purush-o7/r3f-visualized)
> *An interactive learning platform for React Three Fiber — from Three.js fundamentals to production-ready 3D on the web*

Every concept has a **live, interactive 3D demo** you can play with directly in the browser. Covers everything from scene basics to custom shaders, physics with Rapier, post-processing effects, and production deployment.

**Built with:** Next.js 16 · React Three Fiber · Three.js · Drei · Rapier · Leva · Tailwind CSS v4

---

### 💪 [rep-tracker](https://github.com/purush-o7/rep-tracker)
> *A mobile-first gym tracker to log workouts, build streaks, track progress, and reach fitness goals*

Features a 120+ exercise library, routine management, streak tracking with a public leaderboard, a partner system, and progress charts. Backed by a robust Supabase PostgreSQL schema with Row Level Security on every table.

**Built with:** Next.js 16 · Supabase · PostgreSQL · shadcn/ui · Recharts · React Hook Form · Zod

---

### ⚡ [fastapi-starter-kit](https://github.com/purush-o7/fastapi-starter-kit)
> *A production-ready starter kit bridging FastAPI backends with Next.js frontends*

A clean foundation for spinning up full-stack projects fast — FastAPI handles backend logic while Next.js delivers a polished, performant frontend experience.

**Built with:** FastAPI · Next.js · TypeScript
